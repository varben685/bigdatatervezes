# GreenRide beadandó

Ez a repository a **Big Data rendszerek tervezése** tárgy beadandóját tartalmazza.

A feladat a **GreenRide** városi mikromobilitási platform architektúratervezése, amelyhez ASR, ADR és C4 modellek készültek.
